Step 0: 초기 상태Initial State - On-Prem Only

모든 서비스가 On-Prem 데이터센터에서 운영됩니다. 20개 마이크로서비스와 5개 데이터 저장소가 단일 위치에서 실행되는 출발점입니다.All services run in a single On-Prem datacenter. This is the starting point with 20 microservices and 5 data stores.

트래픽 분배Traffic Distribution On-Prem 100% — 모든 요청이 단일 데이터센터로 집중됩니다.All requests are concentrated in a single datacenter.

20개 마이크로서비스 구조20 Microservices Architecture

5개 도메인 x 20개 서비스 아키텍처5 Domains x 20 Services Architecture Architecture

쇼핑몰 플랫폼은 DDD(Domain-Driven Design) 기반으로, 각 도메인이 독립적인 배포 단위와 데이터 저장소를 가지는 Bounded Context 패턴으로 5개 도메인으로 분리됩니다. 각 서비스는 독립적으로 배포 가능한 마이크로서비스이며, 현재는 nginx:alpine 플레이스홀더로 실행 중입니다.

도메인 구성

DomainServicesStackTarget AWS Service
Core product-catalog, inventory, pricing, search Go / Gin DSQL, OpenSearch
User user, notification, recommendation Java / Python DSQL, ElastiCache
Fulfillment order, payment, shipping, delivery-tracking Java / Go DSQL, MSK
Business seller, analytics, promotion, review Python / Java DocumentDB, OpenSearch
Platform api-gateway, event-processor, media, config Go / Python / Java MSK, S3

서비스 통신 패턴

서비스간 동기 호출은 api-gateway를 통한 REST/gRPC로 처리하고, 비동기 이벤트는 MSK (Kafka)를 통해 발행/구독합니다. order → payment → shipping 흐름은 Saga 패턴(분산 트랜잭션을 보상 트랜잭션 체인으로 처리하는 패턴)으로 구현됩니다.

데이터스토어 매핑Datastore Mapping

On-Prem → AWS 데이터스토어 매핑 전략On-Prem → AWS Datastore Mapping Strategy Database

On-Prem의 오픈소스 데이터 저장소를 AWS 매니지드 서비스로 매핑합니다. 핵심 원칙은 Wire-protocol 호환성(클라이언트↔DB 간 네트워크 통신 프로토콜이 동일하여 연결 문자열만 변경하면 됨)을 유지하여 애플리케이션 코드 변경을 최소화하는 것입니다.

On-PremAWS Service호환성마이그레이션 난이도
PostgreSQL 15 Aurora DSQL PostgreSQL 16 wire protocol 중 (DSQL 제약 확인 필요)
MongoDB 7.0 DocumentDB 8.0 MongoDB API 호환 중 (일부 aggregation 미지원)
Kafka 3.x MSK (Kafka 3.x) 100% 호환 하 (브로커 엔드포인트만 변경)
Redis 7.x ElastiCache (Valkey 7) Redis wire protocol 하 (캐시이므로 데이터 손실 무관)
Elasticsearch 8.x OpenSearch 2.x REST API 호환 (대부분) 중 (인덱스 매핑 검증 필요)
DSQL 주의사항 DSQL은 PostgreSQL wire protocol을 지원하지만, 일부 DDL/확장 기능에 제약이 있습니다. 마이그레이션 전 애플리케이션에서 사용하는 PostgreSQL 기능을 점검해야 합니다. 특히 LISTEN/NOTIFY, custom extensions, stored procedures는 지원되지 않을 수 있습니다.
