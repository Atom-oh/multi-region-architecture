프로젝트 개요Project Overview

이 문서는 On-Prem 데이터센터에서 운영중인 쇼핑몰 플랫폼을 AWS 멀티리전 아키텍처로 마이그레이션하는 전체 과정을 다룹니다.This document covers the end-to-end migration process from an On-Prem datacenter to AWS multi-region architecture.

핵심 목표Key Objectives Write-Primary / Read-Local 패턴의 Active-Active 멀티리전 구성. CloudFront + WAF를 통한 글로벌 트래픽 관리. 무중단 마이그레이션을 위한 7단계 점진적 전환.Active-Active multi-region with Write-Primary / Read-Local pattern. Global traffic management via CloudFront + WAF. Zero-downtime migration through 7 incremental steps.
